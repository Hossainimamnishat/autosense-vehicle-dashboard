📋 ALTERNATIVE: Manual Java 17 Installation
================================================================================

If you prefer command-line builds:

1. VERIFY Java 17 installation:
   - Open "Add or Remove Programs" in Windows Settings
   - Search for "Amazon Corretto" or "Java"
   - If not listed → Installation failed

2. REINSTALL Java 17:
   - Download: https://corretto.aws/downloads/latest/amazon-corretto-17-x64-windows-jdk.msi
   - RIGHT-CLICK → "Run as Administrator"
   - Use default installation path
   - Complete the installation

3. RESTART PowerShell (important!)

4. FIND Java path:
   - Run: find-java.bat
   - Or check: C:\Program Files\Amazon Corretto\

5. CONFIGURE gradle.properties:
   - Edit: gradle.properties
   - Add line (replace with actual path):
     org.gradle.java.home=C\\:\\Program Files\\Amazon Corretto\\jdk17.0.13

6. BUILD:
   - Run: .\gradlew assembleDebug
